Technical Definition

A Graph $G$ consists of a non-empty finite set $V(G)$ (vertex set) and a finite set $E(G)$ (edge set) of unordered pairs of elements of $V(G)$

Example


Let $G = (V(G),E(G))$ where

$V(G) = \{ a,b,c,d,e,f \}$

$E(G) = \{ \{a,b\},\{a,e\},\{b,c\},$ $\phantom{E(G) =} \{b,e\},\{c,d\},\{e,d\},\{d,f\} \}$

$\phantom{E(G)} = \{ ab,ae,bc,be,cd,ed,df \}$
